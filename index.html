Find a river access near your location.

This is where you can use a spatial query to find access near you. Steps needed:

  • Add a geocoder... maybe GeoNames, or ESRI's geocoder (does it take tokens?)
  • Geocoder will return a point, use this point as the input for a spatial query. Visualize point on map with a marker.
  • Use the ESRI Leaflet functionality for spatial query.
  • Return the points... will they display the same?
  • Put the points into the side table. Instead of showing river miles, show distance from location.

Create a Float Plan!

This is where you can select a start and end point, and get all of the points in between. They will sorted from upstream to downstream and reported with river miles to make it easy to come up with a quick plan of action. Also, download the point into a format that can go in your GPS!
